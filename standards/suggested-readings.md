# Suggested Readings

When starting out, we encourage devs to have a read through these materials. You don't need to comprehensively read everything just familiarize with the basics. For those with experience, these materials are also good for reviewing the fundamentals or getting updated on new things with these technologies.

## Ruby on Rails

- [Getting Started](https://guides.rubyonrails.org/getting_started.html)
- [Ruby in twenty minutes](https://www.ruby-lang.org/en/documentation/quickstart/)
- [Ruby Styleguide](https://rubystyle.guide/)

## Elixir

- [Getting Started](https://elixir-lang.org/getting-started/introduction.html)
- [Phoenix Documentation](https://hexdocs.pm/phoenix/up_and_running.html)
- [Elixir Styleguide](https://github.com/christopheradams/elixir_style_guide)

## React JS

- [React Documentation](https://reactjs.org/)

## TypeScript

- [TypeScript Documentation](https://www.typescriptlang.org/docs/handbook/typescript-from-scratch.html)

## CSS
